# 📊 Friend Interaction Tracker

## 🚀 Description

A modern web application to track and manage interactions with friends, including calls, text messages, and video chats. It provides a timeline view with filtering options and insightful dashboard statistics to help users stay connected and organized.

---

## 🛠️ Technologies Used

* React.js
* Next.js
* Tailwind CSS
* DaisyUI
* Context API

---

## ✨ Key Features

* 🔍 **Filterable Timeline** — Easily filter interactions by Text, Call, or Video.
* 📈 **Dashboard Analytics** — View total friends, interaction counts, and status insights.
* 👤 **Friend Details Page** — View detailed information about each friend including their interactions and activity history.
